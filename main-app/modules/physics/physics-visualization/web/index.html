基础模块 (Python Matplotlib)

抛体运动

模拟并可视化抛体运动轨迹，可调整初速度和角度。

python ../stage1_basics/projectile.py

单摆

模拟单摆运动，对比小角度近似与数值解。

python ../stage1_basics/pendulum.py

点电荷场线

绘制两个点电荷的电场场线分布。

python ../stage1_basics/electric_field.py

交互式模块 (p5.js / Three.js)

此部分将包含使用 JavaScript 库实现的交互式物理模拟。

波动干涉

使用 p5.js 模拟波的干涉现象。

热扩散

使用 p5.js 或 Three.js 模拟热扩散过程。

磁场分布

可视化磁场分布。

Geogebra 平抛运动演示

使用 Geogebra Classic 5 演示物体平抛运动，可调节平台高度和初始速度。
