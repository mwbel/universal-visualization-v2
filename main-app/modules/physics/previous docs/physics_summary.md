# 普通高中教科书·物理必修 第一册 - 关键公式与定理

## 第 1 章 - 物体运动的描述

### 1.1 运动与质点模型
- **机械运动 (Mechanical Motion):** 一个物体相对于其他物体的位置变化。
- **参考系 (Reference Frame):** 描述物体运动时，被选作参照的物体。
- **质点 (Mass Point):** 用来代替物体的有质量的点，是一种理想模型。当物体的大小和形状在所研究的问题中可以忽略时，可将物体视为质点。
- **位移 (Displacement):** 从初位置指向末位置的有向线段，是矢量。
- **路程 (Path):** 物体运动轨迹的长度，是标量。

### 1.2 & 1.3 怎样描述运动的快慢
- **速度 (Velocity):**
  - 定义: `v = s / t`
  - 描述物体运动快慢和方向的物理量，是矢量。
  - 单位: 米每秒 (m/s)。
- **匀速直线运动 (Uniform Rectilinear Motion):** 物体沿直线运动，如果在相等的时间内通过的位移相等。
- **平均速度 (Average Velocity):**
  - 定义: `v̄ = s / t`
  - 描述物体在某段时间或某段位移内的平均快慢程度。
- **瞬时速度 (Instantaneous Velocity):**
  - 运动物体在某一时刻或某一位置时的速度。
  - **瞬时速率 (Instantaneous Speed):** 瞬时速度的大小。

### 1.4 怎样描述速度变化的快慢
- **加速度 (Acceleration):**
  - 定义: `a = (vt - v0) / t`
  - 描述物体速度变化快慢的物理量，是矢量。
  - 单位: 米每二次方秒 (m/s²)。
  - 方向: 与速度变化量 (Δv) 的方向相同。
- **匀变速直线运动 (Rectilinear Motion with Constant Acceleration):** 物体做直线运动时，如果它的加速度大小、方向都不变。

## 第 2 章 - 匀变速直线运动的规律

### 2.2 匀变速直线运动的规律
- **速度-时间关系:**
  - `vt = v0 + at`
- **位移-时间关系:**
  - `s = v0*t + (1/2)at²`
- **速度-位移关系:**
  - `vt² - v0² = 2as`
- **默顿定理 (平均速度):**
  - `v̄ = (v0 + vt) / 2`

### 2.3 自由落体运动的规律
- **定义:** 物体只在重力作用下从静止开始下落的运动，是初速度为 0，加速度为 `g` 的匀加速直线运动。
- **重力加速度 (Acceleration of Gravity):** `g ≈ 9.8 m/s²`
- **速度-时间关系:**
  - `vt = gt`
- **位移-时间关系:**
  - `h = (1/2)gt²`
- **速度-位移关系:**
  - `vt² = 2gh`

## 第 3 章 - 力与相互作用

### 3.1 重力
- **重力 (Gravity):** 由于地球的吸引而使物体受到的力。
  - `G = mg`
- **重心 (Center of Gravity):** 物体所受重力的等效作用点。

### 3.2 弹力
- **弹力 (Elastic Force):** 发生弹性形变的物体由于要恢复原状，对跟它接触的物体产生的力的作用。
- **胡克定律 (Hooke's Law):** 在弹性限度内，弹力的大小 `F` 跟弹簧伸长（或缩短）的长度 `x` 成正比。
  - `F = kx` (k 为劲度系数)

### 3.3 摩擦力
- **滑动摩擦力 (Sliding Friction):**
  - `f = μN` (μ 为动摩擦因数, N 为正压力)
- **静摩擦力 (Static Friction):** 大小在 0 和最大静摩擦力 `f_max` 之间变化。

### 3.5 & 3.6 力的合成与分解
- **合力 (Resultant Force):** 一个力如果能与几个力共同作用的效果相同，这个力就是那几个力的合力。
- **分力 (Component Force):** 组成合力的那几个力。
- **力的平行四边形定则 (Parallelogram Rule):** 两个共点力的合力，可以用表示这两个力的线段为邻边构成的平行四边形的对角线表示。
- **矢量 (Vector):** 既有大小又有方向，并遵循平行四边形定则合成的物理量 (如位移, 速度, 力)。
- **标量 (Scalar):** 只有大小，没有方向的物理量 (如路程, 质量, 温度)。

### 3.7 共点力的平衡
- **平衡状态 (Equilibrium State):** 物体保持静止或匀速直线运动状态。
- **共点力平衡条件:** 物体所受的合力为零。
  - `F_合 = 0`

## 第 4 章 - 牛顿运动定律

### 4.1 牛顿第一定律 (惯性定律)
- **内容:** 一切物体总保持匀速直线运动状态或静止状态，直到有外力迫使它改变运动状态为止。
- **惯性 (Inertia):** 物体保持匀速直线运动或静止状态的特性。质量是惯性大小的量度。

### 4.3 牛顿第二定律
- **内容:** 物体的加速度跟受到的作用力成正比，跟物体的质量成反比。
  - `F_合 = ma`
- **矢量性:** 加速度的方向始终跟合力的方向相同。

### 4.4 牛顿第三定律
- **内容:** 两个物体之间的作用力和反作用力总是大小相等，方向相反，作用在一条直线上。
  - `F_甲对乙 = -F_乙对甲`
- **特点:** 作用力和反作用力作用在不同物体上，性质相同，同时产生，同时消失。

### 4.6 超重与失重
- **超重 (Overweight):** 物体对悬挂物的拉力（或对支持物的压力）大于物体所受重力的现象。发生在物体有向上加速度时。
  - `N = mg + ma`
- **失重 (Weightlessness):** 物体对悬挂物的拉力（或对支持物的压力）小于物体所受重力的现象。发生在物体有向下加速度时。
  - `N = mg - ma`
- **完全失重:** 当 `a = g` 时，`N = 0`。
