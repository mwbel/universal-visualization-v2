# 可视化微分几何项目执行清单 20251121

## 项目概述
基于《可视化微分几何与形式》教材内容，构建完整的可视化微分几何在线教学平台。该项目包含39章内容，涵盖从基础的欧几里得几何到高级的张量分析等主题。

## 项目架构设计

### 一、技术栈选择
- **前端框架**: React + TypeScript
- **可视化引擎**: Three.js + D3.js
- **数学计算**: Math.js + Numeric.js
- **UI组件**: Ant Design + 自定义组件
- **构建工具**: Vite
- **测试框架**: Jest + Testing Library
