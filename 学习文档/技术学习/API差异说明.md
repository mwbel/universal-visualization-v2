# 🎯 重要发现：API文档与教程的差异说明

## 🔍 您发现了什么？

您发现了API文档中显示的字段与教程中提到的字段不一致，这是一个非常棒的观察！

## 📊 两种API格式对比

### 🆕 **新格式**（当前API文档中显示的）
```json
{
  "prompt": "您的描述",                    // 必需
  "template_id": "normal_distribution",    // 可选：指定模板ID
  "parameters": {                         // 可选：模板参数
    "mu": 2,
    "sigma": 0.5,
    "show_cdf": true
  },
  "user_preferences": {}                  // 可选：用户偏好
}
```

### 📚 **旧格式**（教程中原本写的）
```json
{
  "prompt": "您的描述",        // 必需
  "discipline": "mathematics", // 可选：学科类型
  "style": "plotly"           // 可选：可视化风格
}
```

## ✅ 好消息：两种格式都支持！

经过测试，系统同时支持两种格式：

1. **新格式**：提供更精确的控制
   - 可以指定具体模板
   - 可以精确控制参数
   - 更适合高级用户

2. **旧格式**：简单易用
   - 只需要描述需求
   - 系统自动匹配模板
   - 更适合初学者

## 🎪 现在您可以：

### 使用简单模式：

### 使用精确控制：

1. **软件版本更新**：API会随着版本升级而变化
2. **向后兼容**：好的系统会保持对旧格式的支持
3. **文档滞后**：教程可能不会立即更新
4. **实际测试**：最好的方法是亲自尝试
