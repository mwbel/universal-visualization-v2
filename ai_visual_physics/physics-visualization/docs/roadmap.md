# 物理学可视化项目路线图

## 阶段 1: 基础模块 (已完成)

- **抛体运动**: 使用 Python Matplotlib 模拟抛体运动轨迹，支持初速度和角度调整，生成动画。
- **单摆**: 使用 Python Matplotlib 模拟单摆运动，对比小角度近似与数值解，生成动画。
- **点电荷场线**: 使用 Python Matplotlib 绘制两个点电荷的电场场线分布图。

## 阶段 2: 交互式模块 (待开发)

- **波动干涉**: 使用 p5.js 实现交互式波动干涉模拟，允许用户调整波源参数。
- **热扩散**: 使用 p5.js 或 Three.js 实现交互式热扩散模拟，可视化温度随时间的变化。
- **磁场分布**: 实现交互式磁场分布可视化，例如电流周围的磁场。

## 阶段 3: 高级模块 (待开发)

- **洛伦兹吸引子**: 使用 Python 模拟并可视化洛伦兹吸引子，展示混沌行为。
- **三体问题**: 使用 Python 模拟并可视化三体运动，展示复杂轨道。
- **量子波函数演化**: 使用 Python 可视化一维或二维量子阱中的波函数演化。
- **黑洞透镜**: 使用 Python 或 Three.js 模拟黑洞引力透镜效应。

## Web 界面 (已完成基础结构)

- **index.html**: 项目主页，展示各个模块的链接和简介。
- **styles.css**: 页面样式。
- **app.js**: 页面交互逻辑。
