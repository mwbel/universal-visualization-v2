# 万物可视化 V2 - 快速启动指南

## 🎯 固定主页配置

**主页面地址**: `http://localhost:3000/index.html`

这是万物可视化V2的固定入口页面，采用智能输入+AI Agent架构。

## 🚀 快速启动

### 方法一：使用启动脚本（推荐）
```bash
# 启动V2系统
./start-v2-fixed.sh

# 停止V2系统
./stop-v2.sh
```

### 方法二：手动启动
```bash
# 终端1：启动后端API
cd backend-v2
python3 main.py

# 终端2：启动前端服务
cd frontend-v2
python3 -m http.server 3000
```

## 📋 服务架构

- **前端界面** (端口3000): 智能输入界面
- **后端API** (端口9999): AI Agent系统
- **智能路由**: 自动识别学科并生成可视化

## 💬 使用示例

在前端主页输入以下自然语言描述：

### 数学可视化
- `正态分布 均值0 标准差1`
- `二项分布 n=10 p=0.3`
- `泊松分布 λ=2`

### 天文可视化
- `太阳系行星轨道运动`
- `大熊座星座图 显示恒星名称`
- `月球相位变化 一个月周期`

### 物理可视化
- `抛体运动 初速度20m/s 发射角度45度`
- `简谐振动 振幅2 频率1Hz`
- `自由落体运动 初始高度100m`

## 🔧 系统特性

- ✅ **智能学科识别**: 自动识别数学、天文、物理等学科
- ✅ **自然语言处理**: 支持中文自然语言描述
- ✅ **实时生成**: 秒级生成交互式可视化
- ✅ **多模板支持**: 内置9+专业可视化模板
- ✅ **错误处理**: 完善的错误提示和回退机制
