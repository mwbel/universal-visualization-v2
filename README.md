# 万物可视化 - 智能生成平台

> 🌌 AI驱动的多学科可视化生成系统

## 🚀 快速开始

### 环境要求
- Python 3.8+
- 现代浏览器

### 启动服务

```bash
# 克隆仓库
git clone https://github.com/your-username/universal-visualization.git
cd universal-visualization

# 启动v2.0后端
cd backend-v2
python main.py

# 访问应用
# 新版前端: http://localhost:9999/frontend-v2/
# 原版前端: http://localhost:9999/main-app/
# API文档: http://localhost:9999/docs
```

## 🎯 项目特色

- 🤖 **智能学科识别** - 自动识别数学、天文、物理学科
- 🎨 **极简用户界面** - 专注输入，去除冗余功能
- 🔧 **模块化架构** - 可扩展的Agent系统
- 📱 **多端支持** - 响应式设计，支持移动设备

## 📋 功能概览

### 🎨 学科支持

- **数学**: 概率分布、线性代数、微积分
- **天文**: 行星运动、星座可视化、天球坐标
- **物理**: 力学、电磁学、波动现象

### 🛠️ 核心架构

- **方案A**: 集中式路由架构
- **Agent系统**: 每学科独立处理模块
- **模板引擎**: 统一的可视化生成系统
- **API网关**: 单一入口的智能路由
