# 茅塞顿开 - 高中全科可视化学习平台

## 项目简介
"茅塞顿开"是一个专注于高中全科知识点的可视化学习平台，通过直观的图形化展示和互动示例，帮助高中生轻松理解复杂的学科概念。

## 🎯 项目目标
- 将高中九大学科（语文、数学、英语、物理、化学、生物、历史、地理、政治）的知识点进行可视化
- 提供互动式学习体验，让学生通过操作加深理解
- 建立知识点之间的关联网络，帮助学生构建完整的知识体系

## 📚 涵盖学科
- **文科**：语文、英语、历史、地理、政治
- **理科**：数学、物理、化学、生物

## 🚀 核心功能
- 知识点可视化图解
- 互动式演示和实验
- 智能学习路径推荐
- 练习与测评系统

## 🛠 技术栈
- 前端：React + TypeScript
- 可视化：D3.js + Three.js
- 样式：Tailwind CSS
- 构建工具：Vite

## 📖 项目结构
```
茅塞顿开/
├── src/
│   ├── components/     # 可复用组件
│   ├── subjects/       # 各学科模块
│   ├── visualizations/ # 可视化组件
│   ├── utils/         # 工具函数
│   └── types/         # TypeScript 类型定义
├── public/
└── docs/
```

## 🎨 设计理念
- **简洁直观**：界面设计简洁明了，突出核心内容
- **色彩丰富**：不同学科使用不同的色彩主题
- **互动性强**：提供丰富的交互体验
- **响应式设计**：支持多端设备访问
