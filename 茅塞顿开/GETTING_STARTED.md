# 🚀 茅塞顿开 - 项目启动指南

## 📋 项目概述

"茅塞顿开"是一个专注于高中全科知识点的可视化学习平台，旨在通过直观的图形化展示和互动示例，帮助高中生轻松理解复杂的学科概念。

## 🛠 技术栈

- **前端框架**: React 18 + TypeScript
- **构建工具**: Vite
- **路由**: React Router DOM
- **样式**: Tailwind CSS
- **可视化**: D3.js + Canvas API
- **图标**: Lucide React

## 📦 安装依赖

```bash
# 进入项目目录
cd 茅塞顿开

# 安装依赖
npm install

# 或者使用 yarn
yarn install
```

## 🎯 启动项目

```bash
# 启动开发服务器
npm run dev

# 或者使用 yarn
yarn dev
```

项目将在 `http://localhost:5173` 启动

## 📚 已实现功能

### 🏠 首页
- 项目介绍和特色展示
- 九大学科选择卡片
- 目前仅数学学科可用（其他学科显示"开发中"）

### 📐 数学学科页面
- 沪教版高中数学教材结构展示
- 6个分册的知识点组织
- 学习进度追踪
- 快速访问功能

### 📊 可视化组件

#### 1. 函数图像绘制器 (`FunctionGraph`)
- 支持多个函数同时绘制
- 实时交互和坐标显示
- 支持自定义函数表达式
- 动画效果和函数切换

#### 2. 几何画板 (`GeometryCanvas`)
- 交互式几何图形绘制
- 支持点、线、圆、三角形
- 坐标显示和测量功能
- 预设图形模板

#### 3. 数列动画演示 (`SequenceAnimation`)
- 等差数列和等比数列的动态展示
- 逐步计算和动画效果
- 公式显示和计算过程
- 可调节动画速度

### 📖 知识点详细页面
- 理论要点和公式总结
- 可视化演示
- 例题解析
- 交互练习题
- 学习路径建议

- 平滑的动画过渡效果
- 直观的用户界面
- 键盘导航支持

## 🔧 开发指南

### 添加新的知识点可视化

1. 在 `src/data/mathematics.ts` 中添加知识点数据
2. 在 `src/subjects/math/visualizations/` 中创建可视化组件
3. 在 `KnowledgePointViewer.tsx` 中集成新组件
4. 更新路由和导航

### 添加新的学科

1. 在 `src/utils/subjects.ts` 中添加学科配置
2. 创建对应的学科目录结构 `src/subjects/[subject]/`
3. 实现学科页面和可视化组件
4. 更新首页的学科卡片

### 自定义可视化

项目使用以下技术进行可视化：
- **D3.js**: 用于复杂的数据可视化和图表
- **Canvas API**: 用于几何图形和交互绘制
- **CSS动画**: 用于UI动画和过渡效果
- **React状态管理**: 处理交互逻辑
